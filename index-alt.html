Plaintext Credential Storage → AES-256 Encryption
Context: Legacy Spring Boot financial API storing user credentials in plaintext. Refactored to AES-256 encryption with OWASP-compliant input validation.
UserService.java Before 
// BEFORE: plaintext storage
public class UserService {

    public void saveUser(String username,
                          String password) {
        // Password stored as-is
        db.save(username, password);
    }

    public boolean authenticate(
            String username,
            String password) {
        String stored = db.get(username);
        return stored.equals(password);
    }
}
UserService.java After 
// AFTER: AES-256 + OWASP validation
@Service
public class UserService {

    private final EncryptionService enc;
    private final Validator validator;

    public void saveUser(String username,
                          String password) {
        validator.validateInput(username);
        String encrypted =
            enc.encrypt(password);
        db.save(username, encrypted);
    }

    public boolean authenticate(
            String username,
            String password) {
        String stored = db.get(username);
        return enc.verify(password, stored);
    }
}
✓ AES-256 encryption at rest ✓ Input validation (OWASP) ↑ Injection resistance ↑ Separation of concerns

Callback Hell → Async/Await with Error Handling
Context: Node.js Express route using nested callbacks for sequential database operations. Refactored to async/await with proper error boundaries.
routes.js Before 
// BEFORE: callback hell
app.get('/trip/:id', (req, res) => {
  db.findTrip(req.params.id,
    (err, trip) => {
      if (err) return res.status(500)
        .send(err);
      db.findUser(trip.userId,
        (err2, user) => {
          if (err2) return res
            .status(500).send(err2);
          res.json({ trip, user });
        });
    });
});
routes.js After 
// AFTER: async/await + middleware
app.get('/trip/:id',
  authenticate,
  async (req, res, next) => {
    try {
      const trip = await
        Trip.findById(req.params.id);
      const user = await
        User.findById(trip.userId);
      res.json({ trip, user });
    } catch (err) {
      next(err); // central handler
    }
  }
);
✓ Readable linear control flow ✓ Centralized error handling ↑ Auth middleware added ↑ Maintainability
